# React Portfolio – Reusable Components - lesson1

This project is a personal portfolio built using React, where I created reusable components to showcase my previous projects. Each project is displayed using a custom component that receives dynamic data through props, making the layout flexible and scalable for future updates.

# Technologies Used

* JavaScript (ES6+)
* Tailwind CSS
* HTML5
* CSS3

* Structuring a scalable React application
* Using Tailwind CSS for fast and responsive styling
* Improving UI design and layout skills
